# 🥷 Ninja OCR AI Trainer

一个正在开发中的 AI OCR / Ninja Warrior 训练助手。

---

## 🧠 为什么做这个项目？

我平常经常会在街头或者公园，

和老大爷们一起玩公共训练器械。

像：

- 单杠
- 引体
- 悬挂
- 双杠
- 握力训练

这些东西我一直都挺感兴趣。

后来无意中发现：

# OCR（Obstacle Course Racing）
# Ninja Warrior

这种偏障碍挑战和极限控制的运动项目。

它们和普通健身不太一样。

非常强调：

- 握力
- 摆荡
- 神经控制
- 爆发力
- 疲劳下持续输出
- 身体协调性

我一下就被吸引了。

---

但训练过程中我慢慢发现：

OCR / Ninja 训练其实有很大的学问。

而现在很多 AI 健身工具，

未必真正理解 OCR 和 Ninja Warrior 训练。

它们很多更像：

> “普通健身 GPT”

生成的内容也比较偏传统健身。

所以我就开始尝试：

# 自己做一个偏 OCR / Ninja 方向的 AI Trainer。

---

## ⚙️ 当前已经实现

目前项目已经实现：

- ✅ AI 训练计划生成
- ✅ OCR / Ninja 风格分析
- ✅ 基础训练等级判断
- ✅ 长期记忆系统（memory.json）
- ✅ 历史训练状态读取
- ✅ Node.js + Express 后端
- ✅ dotenv API key 环境变量保护
- ✅ GitHub 版本管理

---

我希望它未来不是：

# 真正像一个懂 OCR / Ninja 的 AI 教练。
